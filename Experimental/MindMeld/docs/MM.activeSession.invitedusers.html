Namespace: invitedusers

MM.activeSession. invitedusers

MM.activeSession.invitedusers represents the InvitedUsers collection in the MindMeld API. The list of User objects who have been invited to join the Session.

Methods

delete(inviteduserid, onSuccess, onFail)

Uninvite the specified user from the specified session

Name Type Description
inviteduserid string

The MindMeld userid of the user to remove from invited user list
onSuccess APISuccessCallback optional

callback for when removing a user from the session was successful
onFail APIErrorCallback optional

callback for when removing a user from the session failed
Example
         function removeUserFromSession () {
            MM.activeSession.invitedusers.delete('<mindmeld user id>', onRemoveUserFromSession);
         }
         function onRemoveUserFromSession (response) {
            // invited user removed from session
         }

get(params, onSuccess, onFail)

Get the list of users that have been added to the invitedusers collection for this session. A request with a user token can get the invitedusers list only if the associated user is permitted to access the session object itself. A request with an admin token can get the invitedusers list for any session associated with your application.

Name Type Description
params QueryParameters optional

query parameters when fetching the invited user list
onSuccess APISuccessCallback optional

callback for when getting invited user list was successful
onFail APIErrorCallback optional

callback for when getting invited user list failed
Example
         function getInvitedUsers () {
            MM.activeSession.invitedusers.get(null, onGetInvitedUsers);
         }
         function onGetInvitedUsers (response) {
            var invitedUsers = response.data;
         }

json(){Array.<Object>}

Helper function returns the JSON data for the invited users list

Example
         function getInvitedUsers () {
            MM.activeSession.invitedusers.get(null, onGetInvitedUsers);
         }
         function onGetInvitedUsers (response) {
            var invitedUsers = MM.activeSession.invitedusers.json();
            // MM.activeSession.invitedusers.json() returns a JSON object
            // containing data received from MM.activeSession.invitedusers.get()
         }

onUpdate(updateHandler, onSuccess, onError, getQueryParams)

Sets the activeSession.invitedusers' onUpdate handler. If no handler is passed in, onUpdate unsubscribes from push events

Name Type Description
updateHandler APISuccessCallback optional

callback for when the activeSession's invited users list updates
onSuccess function optional

callback for when subscription to onUpdate event succeeds
onError function optional

callback for when subscription to onUpdate event fails
getQueryParams QueryParamGetter optional

custom function used to determine QueryParameters used to in get() request when collection updates
Examples

Setting the onUpdate handler, adding a new invited user, and obtaining the latest invited users list 

         function invitedUsersOnUpdateExample () {
            // set the onUpdate handler for the invitedusers list
            MM.activeSession.invitedusers.onUpdate(
                onInvitedUsersUpdate,
                onSubscribedToInvitedUsersUpdates,
                onSubscribeToInvitedUserUpdatesError,
                getInvitedUserListParams
            );
         }
         function onSubscribedToInvitedUsersUpdates () {
            // successfully subscribed to updates to the session's invitedusers list
            // now, invite a new user
            inviteNewUser();
         }
         function onSubscribeToInvitedUserUpdatesError () {
            console.log('error subscribing to invited user list updates');
         }
         function getInvitedUserListParams () {
            // When the invited user list updates, only fetch 5 objects
            return {
                limit: 5
            };
         }
         function onInvitedUsersUpdate () {
            // there was an update to the invitedusers list
            var invitedusers = MM.activeSession.invitedusers.json();
            // invitedusers contains the latest list of invitedusers
         }
         function inviteNewUser () {
            var newInvitedUserData = {
                provider: 'simple',
                userid: 'einstein79',
                name: 'Albert Einstein'
            };
            MM.activeSession.invitedusers.post(newInvitedUserData);
         }

Deregistering the onUpdate handler 

         function deregisterInvitedUsersOnUpdate () {
            MM.activeSession.invitedusers.onUpdate(null);
         }

post(newInvitedUserData, onSuccess, onFail)

Invite a new user to the active session

Name Type Description
newInvitedUserData Object

object specifying userid of user to be added to active session

Name Type Description
provider string

The name of the authentication provider that you are using in your application. This should be 'simple' for Simple User Authentication. For third-party authentication, this should be the name of the third-party provider, such as 'facebook'
userid string

The userid for the user to invite. This should be the user id value provided by your authentication service. This should not be the MindMeld userid
name string

The name of the user to invite
onSuccess APISuccessCallback optional

callback for when adding user to session was successful
onFail APIErrorCallback optional

callback for when adding live user to session failed
Example
         function inviteUser () {
            var newInvitedUserData = {
                provider: 'simple',
                userid: 'einstein79',
                name: 'Albert Einstein'
            };
            MM.activeSession.invitedusers.post(newInvitedUserData, onInviteNewUser);
         }
         function onInviteNewUser (response) {
            // New user invited to session
         }
